Websites

Now, its no secret that alot of websites often include crafty tricks in order to limit the user's liberties within the site, aswell as track every single datapoint that the user happens to expose about themselves. But, if this is so commonly the case, then how do people which websites are safe?

Put simply, you dont!

There is ultimately no way to tell if a website or a website hoster is tracking you or not. In an attempt to make this site more trust worthy, I have made the site completely static, which means that it does not contain any images or javascript, which are two things that are very commonly used as data collection points.
